DummyJSON API(https://dummyjson.com/products)를 사용하여 다음 기능을 구현하세요:
상품 목록 표시: 모든 상품을 카드 형태로 표시 (이미지, 제목, 가격, 평점)

검색 기능: 제목으로 상품 필터링 (실시간 검색)

카테고리 필터: 카테고리 선택시 해당 상품만 표시 (예: 스마트폰, 랩탑)
 
장바구니 기능:

상품 추가/제거 기능

장바구니 아이콘에 총 수량 표시

장바구니 목록 표시 (호버시 드롭다운)

기술 요구사항
**async/await**를 사용하여 API 호출

Fetch API 활용

반응형 디자인 (CSS Grid/Flex)

LocalStorage를 사용한 장바구니 데이터 영구 저장
